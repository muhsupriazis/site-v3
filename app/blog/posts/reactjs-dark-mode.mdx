Dark mode adalah fitur yang semakin populer dalam pengembangan web. Ini tidak hanya membantu mengurangi kelelahan mata pengguna, tetapi juga memberikan tampilan yang lebih modern dan elegan pada aplikasi Anda. Dalam artikel ini, kita akan membahas cara sederhana untuk menerapkan dark mode dalam aplikasi ReactJS.

### Setup Proyek React

Jika Anda belum memiliki proyek React, Anda bisa membuatnya dengan menggunakan Create React App:

```bash
npx create-react-app dark-mode-example
cd dark-mode-example
```

### Membuat Toggle untuk Dark Mode

Pertama, kita perlu membuat komponen toggle untuk mengaktifkan dan menonaktifkan dark mode. Buat file baru `src/DarkModeToggle.js` dan tambahkan kode berikut:

```javascript
import React, { useState, useEffect } from "react";

const DarkModeToggle = () => {
  const [darkMode, setDarkMode] = useState(false);

  useEffect(() => {
    if (darkMode) {
      document.body.classList.add("dark-mode");
    } else {
      document.body.classList.remove("dark-mode");
    }
  }, [darkMode]);

  const toggleDarkMode = () => {
    setDarkMode(!darkMode);
  };

  return (
    <button onClick={toggleDarkMode}>
      {darkMode ? "Switch to Light Mode" : "Switch to Dark Mode"}
    </button>
  );
};

export default DarkModeToggle;
```

### Menambahkan CSS untuk Dark Mode

Sekarang kita perlu menambahkan CSS yang akan diterapkan ketika dark mode aktif. Buka file `src/index.css` dan tambahkan kode berikut:

```css
body {
  transition: background-color 0.3s, color 0.3s;
}

body.dark-mode {
  background-color: #121212;
  color: #ffffff;
}

/* Anda bisa menambahkan lebih banyak gaya sesuai kebutuhan */
```

### Menggunakan Toggle dalam Aplikasi

Terakhir, kita akan menggunakan komponen `DarkModeToggle` dalam aplikasi kita. Buka file `src/App.js` dan modifikasi seperti berikut:

```javascript
import React from "react";
import DarkModeToggle from "./DarkModeToggle";
import "./App.css";

function App() {
  return (
    <div className="App">
      <header className="App-header">
        <h1>Dark Mode Example</h1>
        <DarkModeToggle />
      </header>
      <main>
        <p>
          Ini adalah contoh sederhana dari implementasi dark mode dalam aplikasi React.
        </p>
      </main>
    </div>
  );
}

export default App;
```

### Jalankan Aplikasi

Sekarang Anda bisa menjalankan aplikasi dengan perintah berikut:

```bash
npm start
```

Jika semua sudah berjalan dengan baik, Anda akan melihat aplikasi React dengan opsi untuk beralih antara mode terang dan gelap.

### Kesimpulan

Menerapkan dark mode dalam aplikasi React sangatlah mudah dan memberikan nilai tambah pada pengalaman pengguna. Anda dapat mengembangkan lebih lanjut fitur ini dengan menyimpan preferensi pengguna di `localStorage` atau menghubungkannya dengan tema aplikasi Anda secara keseluruhan.

Semoga artikel ini membantu Anda dalam memahami konsep dasar dan implementasi dark mode dalam ReactJS!
